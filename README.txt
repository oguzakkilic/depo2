== Description ==

Trendyol ücretsiz woocommerce entegrasyon eklentisi ile Trendyol pazaryerindeki mağazanızın işlemlerini tek panelden yapabilirsiniz.

## ÜCRETSİZ ÖZELLİKLER

- Toplu ürün eşleştirme
- Toplu ürün fiyat ve stok güncelleme
- Toplu ürün fiyat değer değiştirme (Yüzde veya sabit fiyat)
- Toplu görsel değiştirme
- Kategori eşleştirme
- Ana marka seçme
- Kargo firması seçme

== Installation ==

Eklentinin kurulumu için 2 yol vardır.

Otomatik Kurulum

1. Wordpress sitenizde "Eklentiler / Yeni Ekle" menüsüne gidin.
2. Sağ kısımdan "Eklentilerde Ara" yazı kutusuna "Woocommerce Trendyol" yazarak eklentiyi bulup "Yükle" ve "Etkinleştir" butonuna basarak kurabilirsiniz.

Manuel Kurulum
1. Bu sayfadaki sağ üstten bulunan "Download" butonua basarak eklentiyi zip halinde indirim.
2. Wordpress sitenizde "Eklentiler / Yeni Ekle" menüsünde gidin.
3. Sayfanın üst kısmında bulunan "Eklenti Yükle" butonuna basın.
4. Açılan panelde "Dosya Seç" butonu basın ve indirdiğiniz dosyayı seçip "Şimdi Kur" butonuna basın ve sonrasında gelen sayfada "Etkinleştir" seçip eklentiyi kullanabilirsiniz.

== Frequently Asked Questions ==

= Woocommerce Trendyol Entegrasyon Eklentisi Ne İşe Yarar? =

Bu eklenti ile wordpress alt yapılı ve woocommerce destekli web sitenizi Trendyol entegrasyonunu yaparak ürünlerinizi yöntebilirsiniz.

= Web Sitemi Trendyol Entegrasyonunu Nasıl Yaparım? =

Eklentiyi kurduktan sonra açılan sayfadan veya websitenizin admin panelinden "Trenyol" menüsünden talimatları izleyerek 20sn içinde entegrasyonu yapabilirsiniz.

= Bu eklentiyi neden kurmalıyım? =

E-ticaret yeni başladıysanız ve bir wordpress siteniz varsa. Tüm pazar yerlerinin panellerini öğrenmeniz gerekecektir. Her pazar yerinin paneli farklı şekilde kullanılmaktadır.
Sizi bu yükten kuratarak zaten alışık olduğunu wordpress panelinden tüm ürünlerinizi tek yerden yönetme imkanı veriyoruz.

= Ürün aktarma bu eklentide var mı? =

Hayır. Ürün aktarma özelliği için ayrı bir eklentimiz mevcutur. İster Trendyoldan sitenize isterseniz sitenizden Trendyola aktarma yapabilirsiniz.
Trendyol ürün aktarma eklentisine ulaşmak için <a href="https://hayatikodla.net/trendyol-woocoommerce-urun-aktarma/">Trendyol Woocommerce Ürün Aktarma</a> linkinden satın alabilirsiniz.

= Ürün listesinde ürün eşlemedi olarak gözüküyor? =

Ürün listesinde sitenizin yavaşalamaması için arkaplanda her 10 dakikada bir ürünleri kontrol edilir. Henüz ürününüz eşleşmediyse 10dk sonra tekrar kontrol etmelisiniz.
